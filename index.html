Electronics Engineer?

Electronics Engineer adalah profesi yang merancang, mengembangkan, dan menguji komponen perangkat elektronik yang digunakan dalam berbagai aplikasi, mulai dari alat rumah tangga, sistem telekomunikasi hingga perangkat medis. Electronics Engineer harus menguasai pemahaman mendalam tentang prinsip-prinsip elektronika, teknik pemrograman, dan kemampuan analitis yang kuat untuk memecahkan masalah teknis memastikan integrasi sistem yang optimal dan inovasi berkelanjutan dalam teknologi elektronik.

Kontribusi?

Kontribusi terhadap teknologi meliputi pengembangan perangkat elektronik yang revolusioner terlibat dalam peningkatan infrastruktur telekomunikasi, termasuk jaringan 5G, WiFi, dan serat optik, serta merancang sistem otomasi dan robotika yang meningkatkan efisiensi industri. berkontribusi pada teknologi energi terbarukan, perangkat medis canggih, dan sistem keamanan elektronik, serta berperan dalam pengembangan teknologi komputasi, kecerdasan buatan, dan transportasi modern.

Membuat sistem pemantauan ketinggian air tandon

Project 2

FireWatch IoT

Sistem FireWatch IoT dirancang untuk memenuhi salah satu tugas saya yang berfungsi untuk pencegahan dini kebakaran dalam ruangan. sistem ini mengunakan 2 microcontroller yang terhubung satu sama lain menggunakan komunikasi CAN-Bus, sistem ini memiliki dashboard untuk memantau parameter tanda tanda munculnya api yang menggunakan Blynk IoT. sistem ini dapat mengirim data dari mikrokontroller ke client menggunakan protokol yang dimiliki oleh blynk sehingga client dapat memantau kondisi dari mana saja, selagi memiliki akses internet.

Project 3

Monitoring daya rumah

Sistem monitoring daya rumah dirancang untuk memenuhi salah satu tugas saya yang berfungsi untuk monitoring tegangan, arus, power, dan lain-lain menggunakan sensor PZEM004T. sistem ini menggunakan dua mikrokontroller slave dan master yang saling terhubung dengan Serial RS485, untuk memonitoring parameter menggunakan user interface website yang menggunakan protokol HTTP Post sebagai protokol untuk mengirim data dari mikrokontroller ke server.
